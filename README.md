# FCC Certificate Projects - Survey Form

## Description

A simple survey form made with HTML and CSS. This is one of the projects required to obtain the Responsive Web Design Certificate from freeCodeCamp.

## Features

**HTML**

-   Form
-   Input Fields
-   Labels
-   Buttons
-   Textarea
-   Radio buttons
-   Checkboxes
-   Dropdowns

**CSS**
